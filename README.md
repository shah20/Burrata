# Burrata
A Food Scheduler

Steps to run application

Install Node<br/>
Install MongoDB<br/>
Install MongoDB Compass(Optional. Used to view DB)<br/>
run npm install<br/>
run MongoDB server (if you are changeing mongoDB server port update port in constants.js file)<br/>
run npm run build command<br/>
